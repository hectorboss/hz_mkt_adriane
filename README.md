# Projeto: Player com Botão de Play (som ativado)

Abra o arquivo `index.html` no seu navegador (pode usar o Live Server do VS Code).

## Como funciona
- Clique no **grande botão de Play** no centro para iniciar o vídeo **com som**.
- Controles inferiores: Play/Pause, Mudo/Desmudo, barra de progresso e tempo.
- O botão central volta a aparecer quando o vídeo termina.

## Arquivos
- `index.html`
- `style.css`
- `script.js`
- `assets/HZ_REELS_ADRIANE.mp4`
